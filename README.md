# VS Code Security Assistant Extension
A specialized VS Code extension developed to enhance secure coding practices for developers.

## 🚀 Overview
This extension acts as a real-time security auditor within the VS Code environment. It scans the active editor for common security pitfalls such as hardcoded credentials, insecure protocols, and potential data leaks.

### Key Features
- **Real-time Pattern Matching:** Uses optimized regex to identify risky code patterns.
- **Developer Alerts:** Provides immediate feedback via VS Code warning messages.
- **Customizable Rules:** Designed to be easily extended with custom security rules and regex patterns.

### Why this matters?
Building tools for the development environment shows a deep understanding of the **Software Development Life Cycle (SDLC)** and a commitment to **DevSecOps** principles.

- **TypeScript**
- **VS Code Extension API**
